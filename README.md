# Review Slider React Project

A professional and modern **Review Slider React Project** built with **React**. This project demonstrates the use of **useState** for interactive navigation between reviews, responsive design, and smooth gradient UI for a modern portfolio look.

## Features

- Built using **React** and functional components
- Uses **useState** to manage the current review index
- **Next** and **Previous** buttons to navigate reviews
- **Circular profile images** with hover zoom effect
- **Responsive design** for mobile and desktop screens
- **Animated gradient background** and floating card style
- Smooth **shadow and hover effects** for professional look
